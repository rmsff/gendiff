## Setup

```sh
$ npm install
```

## If you are using Linux and you have the error 
```sh
$ gendiff
/usr/bin/env: 'node': No such file or directory
```
then enter the command 
```sh
$ sudo ln -s $(which nodejs) /usr/local/bin/node
```
This should solve the problem.
